## Overview

This repository accompanies a research seminar on the **Neyman-Pearson Lemma** and its application to statistical hypothesis testing using the Iris dataset in R.

The project demonstrates the theoretical foundation and practical implementation of hypothesis testing based on the Neyman-Pearson framework, including:

- Simple hypothesis testing
- Neyman-Pearson Lemma
- Likelihood Ratio Tests
- Type I and Type II errors
- Statistical power
- One-sided hypothesis tests
  - Left-tailed tests
  - Right-tailed tests
- Two-sided hypothesis tests
- Decision rules
- Practical implementation in R
- Interpretation of results

The analyses use the classic **Iris dataset** to illustrate each concept through reproducible examples.

## Objectives

The project aims to

1. Explain the Neyman-Pearson Lemma.
2. Demonstrate likelihood ratio hypothesis testing.
3. Perform left-tailed, right-tailed and two-sided hypothesis tests.
4. Compare theoretical and computational approaches.
5. Visualize rejection regions and sampling distributions.
6. Interpret statistical decisions using R.
